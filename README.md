# ASH Global Initiatives

![ASH Global Initiatives](https://ashglobal.ke/logo.png)

## Empowering Communities Through Education, Innovation, Technology & Sustainable Development

### About ASH Global Initiatives

ASH Global Initiatives is a non-profit organization dedicated to empowering individuals, families, and communities through education, digital transformation, entrepreneurship, social welfare, and sustainable development programs.

Our mission is to create opportunities that enable people to rise above poverty, gain practical skills, embrace technology, and build sustainable livelihoods that positively impact future generations.

We believe that every person deserves access to knowledge, opportunities, and resources that can transform their lives and communities.

---

## Our Vision

To become a leading community transformation organization that empowers people through education, technology, innovation, and sustainable development.

---

## Our Mission

To equip communities with practical skills, digital literacy, entrepreneurship opportunities, and social support systems that foster self-reliance, economic empowerment, and sustainable growth.

# Getting Started

## Clone Repository

```bash
git clone https://github.com/ashglobalinitiatives/ashglobal.git
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

## Build Production Version

```bash
npm run build
```

## Run Production

```bash
npm start
```
